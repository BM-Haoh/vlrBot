# Privacy Policy for VlrBot

This Privacy Policy explains how **VlrBot** ("the Bot") handles data. By using the Bot, you agree to the terms described below.

### 1. Data Collection
The Bot does not collect, store, or share any personal data from Discord users (such as names, emails, or private messages). 

### 2. Technical Logs
The Bot uses standard Python logging (`logging.FileHandler`) to record technical events, such as errors and command execution status. These logs are stored locally for debugging purposes and are not shared with third parties.

### 3. Public Valorant Data
All match and team statistics displayed by the Bot are publicly available data retrieved from vlr.gg. The Bot does not claim ownership of this data.

### 4. Data Deletion
Since no personal user data is stored in our PostgreSQL database (which only contains VCT statistics), there is no user-specific data to be deleted.

### 5. Contact
For any questions regarding this policy, please contact the developer via GitHub.
